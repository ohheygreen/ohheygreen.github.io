Meetic | Event Mobile App Design Process

Meetic is a simplified mobile app that includes a reward driven QR code scavenger hunt, which will help increase engagement at events and encourage participation

Type

School project

Role

UX Designer & Researcher

Length

1 month

Tools

Photoshop, Illustrator, XD

Objective

Prototype an app that increases engagement at events by encouraging participation through an interactive Scavenger Hunt

Tasks

User Interview, Market Research, Problem Statement, HMW’s, Wireframe, Information Architecture, User Flow

Our dream team :)

Initial Research

We picked 3 similar event/ food apps that have the same gamification concept to research.
Usability test was then conducted on a few participants to detect the moments of delight and pain points while using these 3 apps in order to pick some features for our app

Problem Statement

After the usability test we found out that users want a simple and informative app with the option for incentives in order to engage with it on a repeat basis.

How Might We

1. How might we ensure the scavenger app is simple to use?
2. How might we reward/ motivate the users in the quest?
3. How might we add payment option?
